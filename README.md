# Docky
Docky is a dockerizer application to help developer run up various dev servers.

Now support only

### WooCommerce
- latest
```bash
bin/docky build woocommerce --version=latest
```

**Database Informations**

DB Name: www  
DB User: root  
DB Password: root

### EC-Cube
- 2.13.3
```bash
bin/docky build eccube --version=2.13.3
```

**Database Informations**

DB Name: www  
DB User: postgres  
DB Password: root
