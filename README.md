# Docky

Docky is a dockerizer application to help developer run up various dev servers.

## Usage

Now support only

### WooCommerce

#### Versions supported

- latest

    ```bash
    bin/docky build woocommerce --version=latest
    ```

**Database Informations**

DB Name: www  
DB User: root  
DB Password: root

### EC-Cube

#### Versions supported

- 2.13.3

    ```bash
    bin/docky build eccube --version=2.13.3
    ```

**Database Informations**

DB Name: www  
DB User: postgres  
DB Password: root
